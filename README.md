# Farah Swilam Portfolio

A simple GitHub Pages portfolio site with an embedded PDF viewer.

## Files

- `index.html` - main portfolio page
- `style.css` - page styling
- `assets/Farah-Swilam-Portfolio-2026.pdf` - embedded portfolio PDF
- `assets/cover.jpg` - cover preview image

## Enable GitHub Pages

1. Upload all files in this folder to the repository root.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Choose branch **main** and folder **/root**.
5. Save.
